In Cell Grid for Microsoft Excel

The XLInCellGrid.xla(m) is an Excel Add-In that allows the user to create very sophisticated Grid reports directly in the cells - ranges, i.e. no frames or dialogs.

Available free for non-commercial use.

• Download XLInCellGrid.xla (Ver. 4.4) - Excel 1997 - 2003

• Download XLInCellGrid.xlam (Ver. 8.3) - Excel 2007 - 2016

 

• Play Flash Demo

 

Features

• Summarize data using customized or built in Excel Formulas

• Explore data by expanding rows or sorting with a simple double click

• All functionality is local to VBA Excel, not 3rd party applications
